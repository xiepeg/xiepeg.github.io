Mybatis中CRUD操作详解

  • 在Mybatis初识中，我们了解了如何快速的代建Mybatis的环境以及简单的select操作。接下来我们将继续学习使用Mybatis来实现数据库的增删改查操作。

一、配置Mybatis环境

1、创建maven项目后，编辑pom.xml文件导入jar包：

    <dependencies>
        <dependency>
            <groupId>mysql</groupId>
            <artifactId>mysql-connector-java</artifactId>
            <version>5.1.48</version>
        </dependency>
        <dependency>
            <groupId>junit</groupId>
            <artifactId>junit</artifactId>
            <version>4.13</version>
            <scope>test</scope>
        </dependency>
        <dependency>
            <groupId>org.mybatis</groupId>
            <artifactId>mybatis</artifactId>
            <version>3.5.5</version>
        </dependency>
        <dependency>
            <groupId>log4j</groupId>
            <artifactId>log4j</artifactId>
            <version>1.2.17</version>
        </dependency>
    </dependencies>

2、定义用户类User和UserDao接口：

  • 在 com.Mybatis.mainclass包下定义User类
package com.Mybatis.mainclass;
import java.io.Serializable;
import java.util.Date;
public class User implements Serializable {
    private Integer id;
    private String sex;
    private String address;
    private String username;
    private Date birthday;
    public Integer getId() { return id; }
    public String getSex() { return sex; }
    public String getAddress() { return address; }
    public String getUsername() { return username; }
    public Date getBirthday() { return birthday; }
    public void setId(Integer id) { this.id = id; }
    public void setSex(String sex) { this.sex = sex; }
    public void setAddress(String address) { this.address = address; }
    public void setUsername(String username) { this.username = username; }
    public void setBirthday(Date birthday) { this.birthday = birthday; }
    @Override
    public String toString() {
        return "User{" +
                "id=" + id +
                ", sex='" + sex + '\'' +
                ", address='" + address + '\'' +
                ", username='" + username + '\'' +
                ", birthday=" + birthday +
                '}';
    }
}
  • 在com.Mybatis.Dao包中定义UserDao接口实现各种增删改查操作：
package com.Mybatis.Dao;
import com.Mybatis.mainclass.User;
import java.util.List;

public interface MyUserDao {
    //查询所有用户
    List<User> findAll();

    //根据id来查询指定的用户
    User findbyId(int id);

    //根据字符串模糊匹配usernamename查询
    List<User> findLikeName(String likename);

    //查询用户总数
    int findCount();

    //增加一个用户
    void addUser(User user);

    //根据id删除一个用户
    void deleteUser(int id)；

    //根据id更新用户信息
    void updateUser();
}

3、定义主配置文件SqlMapConfig.xml

  • 在resources路径下创建SqlMapConfig.xml文件
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE configuration
        PUBLIC "-//mybatis.org//DTD Config 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-config.dtd">
<configuration>
    <!--导入外部配置文件-->
    <properties resource="JDBCConfig.properties" ></properties>

    <!--设置别名，批量别名定义，扫描整个包下的类，
    别名为类名（首字母大写或小写都可以），则在映射文件中无需写全类名-->
    <typeAliases>
        <package name="com.Mybatis.mainclass"></package>
    </typeAliases>
    <environments default="mysql">
        <environment id="mysql">
            <transactionManager type="JDBC"></transactionManager>
            <dataSource type="POOLED">
                <!--导入配置参数-->
                <property name="driver" value="${jdbc.driver}"></property>
                <property name="url" value="${jdbc.url}"></property>
                <property name="username" value="${jdbc.username}"></property>
                <property name="password" value="${jdbc.password}"></property>
            </dataSource>
        </environment>
    </environments>
    <!--指定映射配置文件位置-->
    <mappers>
        <mapper resource="com/Mybatis/Dao/UserDao.xml"/>
    </mappers>
</configuration>

4、定义映射配置文件MyUserDao.xml

  • 在com/Mybatis/Dao下定义MyUserDao.xml配置文件
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper
        PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">

<mapper namespace="com.Mybatis.Dao.MyUserDao">
    <!--在主配置文件中采用了别名，可在resultType中直接写类名-->
    <select id="findAll" resultType="User">
        select * from user
     </select>
</mapper>

二、单表的查询操作

  • 查询操作有四个，分别为
      //查询所有用户
    List<User> findAll();

    //根据id来查询指定的用户
    User findbyId(int id);

    //根据字符串模糊匹配usernamename查询
    List<User> findLikeName(String likename);

    //查询用户总数
    int findCount()

与之对应，我们则需要在MyUserDao.xml下创建四个映射：

<!--在主配置文件中采用了别名，可在resultType中直接写类名-->
    <select id="findAll" resultType="User">
        select * from user
     </select>
    <!--通过#{}可以实现 preparedStatement 向占位符中设置值，
    自动进行 java 类型和 jdbc 类型转换， #{}可以有效防止 sql 注入。-->
    <select id="findbyId" parameterType="int" resultType="User">
        select * from  user where id=#{id}
    </select>

    <select id="findLikeName" parameterType="String" resultType="User">
        select * from user where username like #{str}
    </select>

    <select id="findCount" resultType="int">
        select count(id) from user
    </select>

三、单表的插入操作

  • 定义插入的映射：
<insert id="addUser" parameterType="User">
        insert into user(username,sex,birthday,address)
         values(#{username},#{sex},#{birthday},#{address})
    </insert>

四、单表的更新操作

  • 定义更新的映射：
<update id="updateUser" parameterType="int">
        update user set username='MikeNew' where id=#{id}
    </update>

五、单表的删除操作

  • 定义删除的映射：
<delete id="deleteUser" parameterType="int">
        delete from user where id=#{uid}
    </delete>

六、测试CRUD

  • 为了测试的简便性，这里使用junit来方便测试，同时将创建SqlSession的语句封装起来放在一起减少代码冗余：
public class Mytest {
    private InputStream is;
    private SqlSessionFactoryBuilder builder;
    private SqlSessionFactory factory;
    private SqlSession session;
    private MyUserDao dao;

    @Before
    public void init() throws IOException {
        is= Resources.getResourceAsStream("SqlMapConfig.xml");
        builder=new SqlSessionFactoryBuilder();
        factory=builder.build(is);
        session=factory.openSession();
        dao=session.getMapper(MyUserDao.class);
    }
    @Test
    public void testfindAll(){
        List<User> lst=dao.findAll();
        for(User user:lst){
            System.out.println(user);
        }
    }
    @Test
    public void testfindbyId(){
        User user=dao.findbyId(1);
        System.out.println(user);
    }

    @Test
    public void testfindlikeName(){
        List<User> lst=dao.findLikeName("%老%");
        for(User user:lst){
            System.out.println(user);
        }
    }
    @Test
    public void testaddUser(){
        User user=new User();
        user.setUsername("Mike");
        user.setAddress("Japan");
        user.setBirthday(new Date());
        user.setSex("男");

        dao.addUser(user);
    }

    @Test
    public void testfindCount(){
        int c=dao.findCount();
        System.out.println(c);
    }

    @Test
    public void testdeleteUser(){
        dao.deleteUser(3);
    }

    @Test
    public void testupdateUser(){
        dao.updateUser(8);
    }

    @After
    public void destroy() throws IOException {
        session.commit();
        session.close();
        is.close();
    }
}
